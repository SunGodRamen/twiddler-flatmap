# twiddler-flatmap

Canonical flat text source for Twiddler chord mappings.

## Intent

Treat Twiddler as a constrained input device and maintain a developer-friendly,
version-controlled mapping layer outside the vendor GUI.

## Layers

- maps/: canonical chord -> semantic action maps
- targets/: platform-specific semantic action bindings
- tools/: validation scripts
- exports/: vendor exports and screenshots
- generated/: derived artifacts
